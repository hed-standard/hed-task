## Notes

- `*` next to a Doc II number indicates a partial or indirect match:
  - **Task 6 / CPT ↔ Doc II #26 (AX-CPT):** Doc I has the general CPT; Doc II has the AX-CPT variant specifically. These are closely related but not identical.
  - **Task 49 / Face Processing ↔ Doc II #38 (CFMT):** Doc I's Face Processing/FFA Localizer and Doc II's Cambridge Face Memory Test both target face processing but are different paradigms. Kept as separate entries (rows 49 and 66).

- EEGMany column lists the EML study number(s) where each task paradigm appears. Some EML studies map to multiple tasks (e.g., EML-1 maps to Tasks 26, 27, 29 which all involve semantic/language processing). Some tasks have multiple EML studies (e.g., Task 2/Flanker is the paradigm in EML-8, 9, 10, 11, 12 — all ERN/conflict studies).
